Examples:

1.) 30 Days Peso Challege:

Imagine you are saving money in a piggy bank on Day 1 you put it Php5. Every day after that, you put in Php2 more than the day before. How much did you save after 30 days?

Given:
a1 = 5
d = 2
n = 30

Use: Sn = {n[2a1+(n-1)d]}/2

S30 = { 30 [2 (5) + (30 - 1) (2) ] } / 2
S30 = { 30 [ 10 + (29) (2) ] } / 2
S30 = [ 30 (10 + 58) ] / 2
S30 = [ 30 (68) ] / 2
S30 = 2,040 / 2
30n = 1,020
Therefore, you have save Php1,020 in 30 Days.

2.) Suppose you have an Arithmetic Sequence where your a1 is 11 and your a2 is 3. Find the sum of all 11 terms.

Given:
a1 = 11
d = a2 - a1
d = 3 - 11
d = -8
n = 11

Use: Sn = {n[2a1+(n-1)d]}/2

S11 = {n[2a1+(n-1)d]}/2
S11 = { 11 [2 (11) + (11 - 1) (-8) ] } / 2
S11 = { 11 [ (22) + (10) (-8) ] } / 2
S11 = { 11 [ (22) + (-80) ] } / 2
S11 = [ 11 (-58) ] / 2
S11 = (-638) / 2
S11 = -319


3.) In a Sequence the 1st term is 12 and the 20th term is 50. Find the sum of all term.

Given:
a1 = 12
a20 = 50
n = 20

Use: Sn = [n(a1+an)]/2

S20 = [ 20 (12 + 50) ] / 2
S20 = [ (20) (62) ] / 2
S20 = 1,240 / 2
S20 = 620
