Home Arithmetic Series

Formula:
Sn = [n(a1+an)]/2

If you don't know the last term:
Sn = {n[2a1+(n-1)d]}/2

Component Breakdown

Sn: The sum of the first n terms.

n: The number of terms you are adding together.

a1: The starting value (the first term).

an: The value of the final term in the series.

d: The constant amount added between each step.
